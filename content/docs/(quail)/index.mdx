## Getting Started

Quail makes it easy to connect to your databases, explore your data with natural language, and create beautiful visualizations and dashboards.

### Quick Start Guide

1. [Set up your first database connection](/docs/connections)
2. Ask questions about your data using the [Chat interface](/docs/chat)
3. Create visualizations with the generated [Charts](/docs/charts)
4. Build [Dashboards](/docs/dashboards) to monitor your key metrics

## Key Features

- **Natural Language Data Exploration**: Ask questions about your data in plain English
- **Interactive Dashboards**: Create and share beautiful, interactive dashboards
- **Powerful SQL Editor**: Write and execute SQL queries with schema-aware assistance
- **Data Visualization**: Generate charts and graphs with just a few clicks
- **AI-Powered Insights**: Get automated insights and anomaly detection

## Main Components

- [**Connections**](/docs/connections): Connect to your databases securely
- [**Dashboards**](/docs/dashboards): Create and customize interactive dashboards
- [**Charts**](/docs/charts): Build visualizations from your data
- [**Chat**](/docs/chat): Interact with your data using natural language
- [**SQL Editor**](/docs/sql-editor): Write and execute SQL queries with advanced features
